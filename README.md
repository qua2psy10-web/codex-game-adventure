# ソラと浮島の宝石

空に浮かぶ島々を駆け抜ける、ブラウザ向け3Dアクションゲームです。主人公ソラを操作し、動く浮島、強風、崩れる足場を越えて空の神殿を目指します。

## 遊び方

- PC: `WASD` または矢印キーで移動、`Space` でジャンプ
- スマートフォン: 画面左のスティックで移動、右の `JUMP` ボタンでジャンプ
- 敵へ近づくと、ソラが自動で踏み込み攻撃します

## 主な要素

- PC・スマートフォン対応のレスポンシブ操作
- コイン、金の羽根、ハート、チェックポイント
- 動く浮島、強風、崩れる足場、隠し部屋
- 電子音BGMと効果音
- クリアタイム、収集数、自己ベストのローカル保存

## 開発

```bash
pnpm install
pnpm run dev
```

プロダクションビルド:

```bash
pnpm run build
```

## 技術構成

- React
- Three.js
- Vite

キャラクター、背景、草地・岩肌のテクスチャには、プロジェクト専用に生成したオリジナル素材を使用しています。
